
# OOPS Project

This is a web based application for an online supermarket. Through 
which 
customers can register and order items.

## Installation

Install my-project with npm for frontend part

```bash
  npm install
  cd my-app
  npm start
```
    
## Tech Stack
- HTML
- CSS
- JavaScript
- ReactJS
- Java
- SpringBoot
- MySQL
- JDBS
